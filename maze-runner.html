Collect the dots. Dodge the bugs. Learn how code changes the game.

Score: 0
Lives: 3
Level: 1
Use arrow keys, WASD, or the buttons. Can you beat the bugs?

Program This Game

Students can open this file in VS Code and change the game live.

  • Change PLAYER_SPEED to make the hero faster.
  • Change GHOST_SPEED to make the bugs easier or harder.
  • Edit GAME_MAP to design a new maze.
  • Change colors in :root to match your own style.
  • Change messages in winMessages.

Map key: # wall, . dot, o power dot, P player, G bug.
